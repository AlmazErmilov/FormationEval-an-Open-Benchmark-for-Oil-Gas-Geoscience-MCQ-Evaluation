# Contributing

Contributions to FormationEval are welcome. Here are some ways you can help:

## Verifying questions

Reviewing existing questions for correctness is valuable. If you find errors:
- Check that the marked answer is correct
- Verify the rationale supports the answer
- Confirm distractors are plausible but clearly wrong
- Open an issue describing the problem and suggested fix

## Adding new questions

New questions are welcome in **English**, **Norwegian**, or **Russian** if they follow these rules:

**Allowed sources:**
- Open-licensed materials (CC BY, CC BY-SA, public domain)
- Government or institutional open data
- Your own original questions based on domain knowledge

**Concept-based questions:**
- Write questions from scratch based on concepts you understand
- Do not copy or closely paraphrase from copyrighted sources
- Include proper attribution in the `sources` field

**Not allowed:**
- Verbatim questions from copyrighted textbooks or exams
- Close paraphrases of copyrighted material
- Questions without clear source attribution

## Schema compliance

All questions must follow the schema in README.md:
- 4 choices with exactly one correct answer
- Required fields: id, question, choices, answer_index, rationale, sources, etc.
- Difficulty label (easy, medium, hard)
- Domain and topic tags

## How to contribute

1. Open an issue to discuss your contribution
2. Fork the repository
3. Make your changes following the guidelines above
4. Submit a pull request with a clear description

## Questions

If you have questions about contributing, open an issue and we will help.
